# dsa
A repository for Data Structures and Algorithms using java 17


# Running Java files

Bash

echo 'export CLASSPATH=$CLASSPATH:.:/workspaces/dsa/algs4.jar' >> ~/.bashrc && source ~/.bashrc

Always navigate to the file's directory first. Do not run from the root.

Navigate: cd src/your specific folder

Compile: javac YourFile.java

Run: java YourFile

Using Input Files To pipe a text file into your program (standard input):

java YourFile < input.txt
